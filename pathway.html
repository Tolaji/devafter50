Learning Activity

This Python code prompts the user for input related to cookies, age, egg cartons, and more. It performs basic mathematical operations and displays results, demonstrating how to interact with user-provided data.

Meal Price Calculator

This Python code calculates the total price of meals, drinks, and appetizers for both children and adults. It then adds tax, tip, and displays a detailed receipt with formatted currency values. The program is interactive, collecting user inputs for various parameters.

Area of Shapes

This Python code calculates the areas of square, rectangle, and circle based on user-provided dimensions. It displays the results and provides prompts for input. The program focuses on core mathematical calculations.

Comparing Numbers

This Python code compares two numbers provided by the user and determines their relationship (greater, equal, or not). It also checks if the user's favorite animal matches a predefined one ("dog") and gives a response accordingly.

Qualifying for Loan

This Python code prompts the user to rate their loan size, credit history, income, and down payment. It then evaluates these inputs to determine if the loan is approved based on predefined criteria. The result is displayed to the user.

Grade Calculator

This Python code calculates and assigns a letter grade based on the provided percentage. It then provides feedback on whether the student passed or failed the course. The program is interactive and straightforward.

Adventure Game

This Python code is an interactive text-based adventure game. It offers multiple levels with scenarios and choices, and the player's decisions determine the outcome. The game involves various paths, obstacles, and rewards, providing a dynamic and engaging experience for the player.

Catch the Food Game

This Python code utilizes Pygame to create a game named "Catch the Food." It involves a player, falling food, and motion-based controls. The game loop manages user inputs, updates game elements, and displays them on the screen.

Guess my Number Game

This Python code implements a guessing game where the user tries to guess a randomly generated magic number. It provides feedback on guesses, tracks the number of attempts, and calculates the time taken. It also offers different levels of performance.

Word Guessing Game

This Python code creates a word guessing game. It prompts the user to choose the number of levels, a category, and then lets them guess words. The game includes power-ups, time limits, and scores.

This Python code is a word guessing game. It prompts the player to guess a word from a predefined list. The player can guess single letters or the entire word. Hints and feedback are provided, and the number of attempts is tracked. When the word is correctly guessed, the player is congratulated.
